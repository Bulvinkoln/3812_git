# Коды для Github
-u (upstream) origin позволяет зафикисровать репозиторий на сервере 
git remote -v показывает fetch (похож на pull) и push

Все инструкции также есть в самом Github при добавлении нового репозитория

Чтобы помочь кому то с проектом на GItHub нужно:

- сделать форк (Fork) какого то интересующего нас репозитория
- сделать git clone для нашей версии этого репозитория
- произвести ветку с предлагаемыми изменениями
- производить изменения только в этой ветке
- отправить эти изменения на свой аккаунт (push)
- отправить pull request в окне на GitHub
