Чтобы скопировать чужой репозиторий и добавить себе, используйте кнопку forked.
Команда git clone создает клон репозитория на ПК.
Команда git pull позволяет скачать все из текущего репозитория и автоматически сделать merge с нашей версии.
Команда git push  позволяет отправить нашу версию репозитория на внешний репозиторий.
